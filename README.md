# SNA-R #
This repository contains some simple examples of the use of Social Network Analysis using R

# EU Cluster for Food Traceability and Trust #
The dataset used for this example contains the Horizon funded projects of the EU Cluster for Food Traceability and Trust and their consortiums. The social network analysis shows the interconnection of projects and consortiums.

This information was filtered from a larger dataset containing all Horizon Europe projects. This dataset titled, CORDIS - EU research projects under HORIZON EUROPE (2021-2027), can be found here:
https://data.europa.eu/data/datasets/cordis-eu-research-projects-under-horizon-europe-2021-2027?locale=en

More information on the cluster can be found at:
https://theros-project.eu/our-cluster/
